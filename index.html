NEXT.JS • TYPESCRIPT • JEST • FIREBASE

CVNNECT

CVNNECT is a serverless SaaS website for creating and sharing digital business cards. It allows users to customize card templates and share with others through QR code, email, or link. Authentication is done through firebase auth with Google and Email as providers. The digital business cards are stored in Cloud firestore which handles basic CRUD operations. In the future I plan to add the ability to add a picture to each card as well as more card templates.

NEXT.JS • TYPESCRIPT • TAILWINDCSS • POSTGRESQL

MovieFinder

MovieFinder allows users to find a random movie at the click of a button, search movies by genres, keywords, release year, or rating. I built this project as I've always had trouble deciding which movies to watch. Users can create an account implemented through NextAuth and favourite movies. It utilizes server side rendering to create a swift user experience. The backend is hosted on a PostgreSQL database and is integrated to the frontend with Prisma and is deployed on Vercel.

MONGODB • EXPRESS.JS • REACT • NODE.JS

BlogPost

Inspired as a Reddit redesign, BlogPost allows users to signup and create posts consisting of text or pictures. The main landing page displays each post and can be sorted by date posted. Users can like posts as well as click on each post to see the full post along with comments made by other users. This project taught me how to create a full stack application from a mockup to a fully integrated frontend and backend. Additionally, I worked with a NoSQL database in MongoDB for the first time.

REACT NATIVE • EXPO • JAVASCRIPT

Tasker

Tasker is a to-do mobile application designed for users to set tasks and goals in a sleek user friendly interface. Users can categorize their tasks and view any upcoming tasks on a calendar view. It is built with React Native and Expo allowing for deployment native to iOS and Android. This project taught me many core concepts in React Native such as props, components, and hooks. I also learned how to integrate multiple external React Native libraries for a unified and clean user experience.

PYTHON • PYCORD

StudyBot

StudyBot is a discord bot that allows you and your friends to stay on track of your work and increase productivity. It includes features such as creating a reminder, reminding other users on the server, starting a study session, creating flashcards, polls, todo lists, and private study rooms. The bot was built with Pycord an API wrapper for Discord. I built the bot after having trouble remaining focused on various coding projects, and as I was frequently on Discord I decided to integrate a productivity/study bot within the app.

SOCKETIO • JAVASCRIPT • HTML • CSS

ChatApp

ChatApp is a web-based messaging application designed for users to connect and communicate to eachother in real-time. Users can choose a display name and interact in a chat room with up to 200 saved messages. It was built using Socket.IO and utilizes Node.JS, Express.JS as backend technologies. This project taught me how to setup an application with a server and client, as well as core frontend concepts involving HTML and CSS, such as flexbox, DOM manipulation, and integrating the backend with the frontend.
